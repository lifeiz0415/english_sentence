# English Sentence Trainer

365개의 핵심 영어 문장을 카드 학습 + 퀴즈 + 음성 재생으로 반복 학습할 수 있는 단일 파일 웹앱입니다.

이 앱은 별도 프레임워크 빌드 없이 `index.html` 하나로 동작하며, 학습 진도는 브라우저에 자동 저장됩니다.

---

## 어떤 앱인가요?

이 프로젝트는 “짧고 자주” 학습하는 데 초점을 둔 영어 문장 트레이너입니다.

- 영어 문장과 한글 뜻을 한 쌍으로 학습
- 퀴즈 모드에서 직접 타이핑하며 암기 점검
- 음성(TTS)으로 **영어 → 한글 뜻** 순서 재생
- 자동재생으로 마지막 문장까지 연속 반복

---

## 주요 기능

### 1) 카드 모드 / 퀴즈 모드
- **카드 모드**: 영어 문장을 보고 뜻(한글) 표시/숨기기
- **퀴즈 모드**: 한글 뜻을 보고 영어 문장을 직접 입력해 정답 확인

### 2) 음성 학습
- 현재 문장을 재생하면:
  1. 영어 문장 읽기
  2. 같은 문장의 한글 뜻 읽기
- 첫 진입 시 첫 문장을 자동 재생
- 문장 이동(이전/다음/랜덤/사이드바 선택) 시 새 문장 자동 재생

### 3) 자동재생
- 버튼: `자동재생 / 자동재생 중지`
- 동작:
  - 영어 읽기 → 한글 뜻 읽기 → **0.1초 대기** → 다음 문장
  - 365번째 문장까지 반복 후 자동 종료
- 자동재생 중 현재 문장을 따라 사이드바가 자동 스크롤됩니다.

### 4) 검색 및 문장 탐색
- 검색창에서 영어/한글 모두 검색 가능
- 정규화 검색(대소문자/기호 영향 최소화)
- 검색 결과 리스트에서 원하는 문장 클릭 이동

### 5) 학습 상태 관리
- **암기 표시**, **중요(★) 표시** 지원
- **진도 초기화** 버튼으로 전체 상태 리셋
- 진도(암기율) 퍼센트 표시

---

## 퀴즈 채점 방식

입력값은 정규화(소문자화 + 구두점 제거) 후 채점됩니다.

- 빈 입력: 입력 안내
- 완전 일치: 정답 + 암기 완료 처리
- 부분적으로 매우 유사: “거의 맞음” 피드백
- 그 외: 오답 피드백

## 데이터/저장 방식

- 기본 문장 수: **365개**
- 문장 포맷: `English — Korean` 또는 `English - Korean`
- 브라우저 LocalStorage 키: `const-english-sentences-v4`

저장 규칙:
- 저장된 데이터가 있고 길이가 기본 문장 수 이상이면 저장 데이터 사용
- 그렇지 않으면 기본 문장 목록으로 초기화

## 사용 팁

- 자동재생으로 전체 문장을 한 바퀴 돌린 뒤,
  퀴즈 모드에서 틀린 문장 위주로 다시 학습하면 효과가 좋습니다.
- 검색으로 특정 주제/표현을 빠르게 모아 반복 학습하세요.
- 암기(완료) 표시를 활용하면 진도 관리가 훨씬 쉬워집니다.
